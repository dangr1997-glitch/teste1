Base Pública para Agente Microsoft Copilot

Este repositório tem como objetivo servir de fonte pública de instruções para o Microsoft Copilot atuar como agente na geração de arquivos JSON para casos jurídicos.

Como funciona

  • O agente deve gerar um arquivo JSON seguindo exatamente o modelo e as regras descritas em Prompt.txt.
  • Não são permitidas explicações ou alterações na estrutura do JSON.
  • A classificação de assuntos e subassuntos deve respeitar as listas contidas no Prompt.txt.
  • O preenchimento dos campos deve seguir as orientações detalhadas.

Arquivos principais

  • Prompt.txt: Instruções completas para preenchimento dos arquivos JSON.

Colaboração

Sugestões de melhorias, novos exemplos ou dúvidas podem ser enviadas via Issues ou Pull Requests.

Objetivo:
Garantir que o Copilot e outros agentes de IA tenham uma fonte de referência padronizada para atuação em tarefas de automação jurídica.
